Using CPAN
          
          * command line client: cpan
          
          * perldoc cpan
          * or man cpan
          
          * basic usage:
          * $> cpan Foo::Bar            <---- installs Foo:Bar
          
          * CPAN shell
          * $> cpan                     <---- start the CPAN.pm shell
          * usually:
            * cpan[1]> install Foo::Bar    <---- installs Foo::Bar
            * cpan[2]> i /FooBar/          <---- search for a module with 
                                                 "FooBar" in its name
